(static) addOrInsertTags(tagArray, forSongId, task) → {promise}

This method inserts newly created tags into the database and saves those new tags into the song's tags sub-document array. It also saves the song onto the newly create tag's subdocument array of songs. Per its name, this method also finds any already existing tags and saves them onto the song if it doesn't yet have that tag. And it also saves the song onto the pre-existing tag. A Fawn task is passed in to ensure atomicity when adding songs to tags and tags onto songs.
Parameters:
Name Type Description
tagArray Array.<string> An array of tag ids (can be a string or integer)
forSongId string | number A string or number representing the song's ordinal number.
task object A Fawn task for keeping updates to the db atomic.

This method inserts a new tag from with an associated song document. This method is normally used to add a new tag that doesn't yet exist to a song directly. The newly inserted tag will get a default associated image name based on the inserted tag name.
